# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1. **Reader** — Someone who tracks their own books, shelves, tags, and reading progress in ReadWell.

2. **Club organizer** — Someone who sets up shared reading plans and suggested titles so a small group can follow along (lighter-weight than a full social network).

## Draft User Stories

1. As a Reader, I want to see all of my books in one library view, so that I can understand my collection at a glance.

2. As a Reader, I want to add a new book with title, author, and reading status, so that I can record what I want to read or am currently reading.

3. As a Reader, I want to edit an existing book’s details, so that I can fix mistakes and update my progress.

4. As a Reader, I want to delete a book I no longer need in my tracker, so that my library stays accurate.

5. As a Reader, I want to create shelves and assign books to them, so that I can group titles by semester, mood, or project.

6. As a Reader, I want to add multiple tags to a book, so that I can find related titles across shelves.

7. As a Reader, I want to filter and sort my library by status, shelf, or tag, so that I can quickly choose what to read next.

8. As a Reader, I want to add or edit a book in a modal without leaving the library page, so that I can stay in context while updating my list.

9. As a Reader, I want to see a small reading summary on the same page as my library, so that I can notice patterns without navigating elsewhere.

10. As a Reader, I want to be taken to a clear confirmation or detail view after I create a book, so that I know the save succeeded.

11. As a Club organizer, I want to create a reading group with a name and short description, so that my friends know what we are doing together.

12. As a Club organizer, I want to attach suggested books to a group, so that members can align on shared picks alongside their personal shelves.
